DIVING DOWN overload SYSTEM

Pequeño proyecto académico enfocado en aprender y experimentar con las bases de la programación para la web, utilizando HTML, CSS y JavaScript como núcleo del desarrollo.

Narrativa: En un sistema colapsado por una sobrecarga masiva de solicitudes, una unidad de información adquiere conciencia y decide descender por las capas del stack web para restaurar el flujo.

Desde la estructura HTML hasta el núcleo del código, el jugador deberá comprender y dominar cada nivel del sistema para corregir el error que lo corrompe

Gameplay: El juego se basa en explorar distintas áreas fundamentales del desarrollo web.
Cada mundo representa una capa del stack, incorporando nuevas mecánicas que reflejan las capacidades reales de cada tecnología:

HTML — La estructura.

CSS — La presentación.

JavaScript — La lógica.

Red — El flujo.

IDE — El origen.

A medida que el jugador avanza, obtiene mayor control sobre el entorno, reflejando el dominio progresivo de cada tecnología.

Objetivos: Eh... ¿Aprender a usar css y javascript? No se es la primera vez que hago una formalidad asi no me pidan mucho, de cosa le ando rogando a ChatGPT para que me escriba lo de arriba de manera bonita y entendible 

Tecnologias Utilizadas: HTML, CSS y JS
